本报告详细评估了隐式求解器在处理范德波尔（Van der Pol）振子时的性能，涵盖了从非刚性（$\mu=1$）到强刚性（$\mu=1000$）的参数范围。测试重点比较了向后微分公式（BDF）、隐式 Runge-Kutta 法（Radau IIA）以及 Rosenbrock 方法。

### 1. 实验设置

*   **测试模型**：范德波尔振子，标准形式：
    $$\dot{x}_1 = x_2, \quad \dot{x}_2 = \mu(1-x_1^2)x_2 - x_1$$
*   **参数范围**：$\mu \in \{1, 10, 100, 1000\}$，反映了从平滑振荡到显著弛豫振荡的过渡。
*   **参评求解器**：
    *   **BDF**：SciPy 实现的变阶隐式多步法。
    *   **Radau**：SciPy 实现的 5 阶隐式 Runge-Kutta 法（Radau IIA），用于高精度基准。
    *   **Rosenbrock (ROS2)**：手动实现的 2 阶 L-稳定半隐式单步法。
*   **参考解**：采用任务 [1_4_1_1] 生成的高精度基准轨迹。
*   **评价指标**：函数调用次数 ($N_{fev}$)、全局平均误差 (Mean Global Error)、执行时间。

### 2. 数值实验结果

下表汇总了不同刚性系数 $\mu$ 下，各求解器在达到中等精度时的典型性能指标：

| $\mu$ | 求解器 | 函数调用次数 ($N_{fev}$) | 平均误差 (Mean Err) | 执行时间 (s) | 特性描述 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **1** | BDF | ~1,200 | $1.7 \times 10^{-3}$ | 0.02 | 效率极高，适合低刚性 |
| | Radau | ~2,500 | $5.0 \times 10^{-5}$ | 0.03 | 精度优势明显 |
| | ROS2 | 5,000 | $1.4 \times 10^{-4}$ | 0.02 | 稳定但对步长敏感 |
| **100** | BDF | ~2,800 | $2.8 \times 10^{-2}$ | 0.05 | 步数随刚性增加缓慢 |
| | Radau | ~15,000 | $4.2 \times 10^{-4}$ | 0.15 | 维持高精度，但开销增大 |
| | ROS2 | 20,000 | $1.0 \times 10^{2}$ | 0.10 | 固定步长无法追踪快变跳变 |
| **1000** | BDF | ~7,100 | $1.3 \times 10^{-2}$ | 0.10 | **刚性场景下的优选** |
| | Radau | ~14,000 | $5.0 \times 10^{-6}$ | 0.12 | 精度极高，适合科研级模拟 |
| | ROS2 | 20,000 | > $10^3$ | 0.08 | 不具备自适应性导致在跳变点失效 |

### 3. 工作-精度曲线 (Work-Precision Analysis)

通过对各 $\mu$ 值生成的 `work_precision_mu_X.png` 图像分析得出以下结论：

1.  **自适应步长的必要性**：随着 $\mu$ 的增加，范德波尔振子呈现出极端的“快-慢”交替特性。隐式方法虽能保证稳定性，但只有具备自适应步长（如 BDF 和 Radau）的算法才能在慢变阶段大幅跨越，并在快变跳变点（Jump point）自动缩小步长以捕捉动力学细节。
2.  **BDF vs. Radau**：
    *   **BDF** 在处理 $\mu=1000$ 时的 $N_{fev}$ 显著少于 Radau，反映了多步法在计算开销上的优势。
    *   **Radau** 在相同 $N_{fev}$ 下提供了更高的精度，适合对数值噪声敏感的混沌或高维系统分析。
3.  **Rosenbrock 表现**：本实验实现的固定步长 ROS2 在低 $\mu$ 时表现良好，但在强刚性情形下，由于缺乏对极限环突变部分的自适应捕捉能力，其全局误差迅速发散。这证明了在强刚性 ODE 系统中，**自适应控制比稳定性本身同样重要**。

### 4. 结论与建议

*   **对于工程应用 ($\mu \le 100$)**：推荐使用 **BDF**，它在保持计算效率的同时能提供稳定的收敛。
*   **对于高精度科学研究**：推荐 **Radau**，其 5 阶精度在处理刚性极限环时表现出卓越的轨道追踪能力。
*   **对于极度刚性系统**：Rosenbrock 类方法（如 ROS34）若配备自适应控制器，将是极佳的选择，因为它不需要非线性方程迭代，单步效率高于 BDF。
