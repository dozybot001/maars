# Markdown Report Templates

## Summary Report

```markdown
## Summary
[2-3 sentence executive summary]

## Key Findings
- Finding 1
- Finding 2
- Finding 3

## Conclusions
[1-2 paragraphs]
```

## Comparison Report

```markdown
## Comparison Criteria
[List criteria]

## Comparison Table
| Criterion | A | B |
|-----------|---|---|
| ...       |   |   |

## Pros and Cons
**A**: Pros: ... Cons: ...
**B**: Pros: ... Cons: ...

## Recommendation
[Clear recommendation with rationale]
```

## Analysis Report

```markdown
## Methodology
[How the analysis was done]

## Results
[Key results, tables, figures]

## Findings
[Interpretation]

## Implications
[So what? Next steps?]
```

## Literature Synthesis Report

```markdown
## Overview
[Scope and sources summary]

## Key Themes
### Theme 1
[Synthesized findings across sources]

### Theme 2
[Synthesized findings across sources]

## Gaps
[What is missing or under-researched]

## Conclusions
[Implications and recommendations]

## References
[Attribution when available]
```

## Technical Report

```markdown
## Problem
[Statement of the problem]

## Approach
[Methodology, design]

## Results
[Data, tables, figures]

## Discussion
[Interpretation, limitations, future work]
```
