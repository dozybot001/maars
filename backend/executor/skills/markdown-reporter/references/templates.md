# Markdown Report Templates

## Summary Report

```markdown
## Summary
[2-3 sentence executive summary]

## Key Findings
- Finding 1
- Finding 2
- Finding 3

## Conclusions
[1-2 paragraphs]
```

## Comparison Report

```markdown
## Comparison Criteria
[List criteria]

## Comparison Table
| Criterion | A | B |
|-----------|---|---|
| ...       |   |   |

## Pros and Cons
**A**: Pros: ... Cons: ...
**B**: Pros: ... Cons: ...

## Recommendation
[Clear recommendation with rationale]
```

## Analysis Report

```markdown
## Methodology
[How the analysis was done]

## Results
[Key results, tables, figures]

## Findings
[Interpretation]

## Implications
[So what? Next steps?]
```
